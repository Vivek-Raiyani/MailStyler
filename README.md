## Installation
1. Clone the repository:
```
   git clone https://github.com/username/repo.git
```
2. Navigate to the project directory:
```
   cd MailStyler
```
3. Install dependencies:
   ```
   $ pip install -r requirements.txt
   ```
4. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
